Étaps personnalisé sur le Kanban

Étape 1: Configurer l'affichage des étapes sur le kanban

Pour afficher le nombre de tickets par étapes sur la vue kanban, il faut aller dans le module Assistance ‣ Menu Configuration ‣ Etapes et choisir une étape puis activer le bouton Show in kanban

À la fin, nous pouvons choisir le menu vue d'ensemble pour voir le résultat

This module will allow you to automatically add the number of tickets per stage on each support team in the kanban view.

Custom stage on kanban

Step 1: Configure the display of stages on the kanban

To display the number of tickets by stages on the kanban view, go to the Helpdesk module ‣ Configuration menu ‣ Stages and choose a stage then activate the button Show in kanban

At the end we can choose the overview menu to see the result
